TDA_PS_sample_images: This folder has binary images from which TDA-based Persistence Statistics features are generated.

You don't need to know what these features are right now

persistence_statistics_1D code.ipynb: This is the code that generates the features. 

Change the file location for images and the csv file location in the end

sample_features:  This is the csv feature file that gets generated, you can look at the features

py37.yaml:  The environment file

Use: conda env create -f my_conda.yaml
